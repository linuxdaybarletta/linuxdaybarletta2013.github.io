La partecipazione è completamente gratuita ed aperta a tutti, ti chiediamo solo di compilare un piccolo modulo di iscrizione e di seguire delle semplici istruzioni per aiutarci ad accoglierti al megio. Grazie!

Cosa portare da casa

Nel caso voglia portare il tuo computer desktop ("fisso"), dovrai necessariamente portare anche monitor, mouse, tastiera ed altre periferiche (se le hai) quali webcam, scheda audio esterna e così via. Non devi invece portare stampanti, modem o penne USB per la navigazione Internet (vanno invece benissimo quelle WiFi).

Se possiedi un computer con lettore DVD, ti verrà fornito un disco necessaro per procedere con l'installazione del tuo nuovo sistema operativo GNU/Linux. In caso invece il tuo computer non sia provvisto di lettore ottico, dovrai portare da casa una memoria USB vuota da non meno di 2 GB ed eseguire il download del file immagine, necessario all'installazione, sul tuo computer (lo stesso che porterai sabato 26 ottobre!).

Sono disponibili diversi file immagine a seconda delle esigenze.

Sistemi a 64 bit

Se hai un computer con architettura a 64 bit, utilizza uno di questi link per il download dell'immagine disco necessaria all'installazione di Linux Mint sul tuo computer.

Se non sai cosa significa "computer a 64 bit", continua a leggere.

Sistemi a 32 bit

Se hai un computer con architettura a 32 bit o non hai idea di quale sia l'architettura del tuo computer, utilizza uno dei link presenti in questa pagina per scaricare l'ultima versione di Linux Mint.

Se te la senti, segui anche le istruzioni che seguono su come realizzare un supporto USB di installazione: ci farà risparmiare molto tempo!

Realizzare una penna USB con Linux Mint

Per prima cosa scarica Linux Live USB creator da questa pagina ed installalo. Una volta avviato il programma, bisogna seguire pochi semplici passi:

  1. seleziona la chiavetta da utilizzare;
  2. seleziona il file immagine che hai scaricato in precedenza (clicca su "ISO/IMG/ZIP" e segui le istruzioni);
  3. spunta "Formatta la chiavetta in FAT32";
  4. clicca sull'icona che rappresenta un fulmine per portare a termine l'operazione.

Preparazione del computer

Per agevolare il lavoro di chi ti aiuterà durante l'installazione del tuo nuovo sistema operativo GNU/Linux a fianco del sistema operativo già installato sul tuo computer, ti invitiamo a svolgere alcune semplici operazioni che ti faranno risparmiare moltissimo tempo ed eviteranno gravi problemi.

Se non hai interesse a mantenere il sistema operativo già presente sul tuo computer o possiedi un computer nuovo privo di sistema operativo e dati interessanti, salta questa sezione.

  1. Per prima cosa esegui un backup di tutti i dati su un hard disk esterno o su un altro computer;
  2. esegui la deframmentazione del disco (trovi informazioni su come fare qui per Windows XP, qui per Windows Vista e qui per Windows 7);
  3. ridimensiona la partizione contenente il sistema operativo già installato a seconda delle tue necessità, lasciando almeno 10 GB liberi (se non sai come fare o riscontri problemi verrai aiutato durante l'installazione, ma l'operazione richiede un certo tempo, prova a cercare qualche informazione prima di rinunciare, non è difficile!).
